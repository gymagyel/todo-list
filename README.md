# Todo List

A modular todo list application built with vanilla JavaScript as part of The Odin Project curriculum.

## Features

- Create multiple projects
- Default Inbox project
- Add and delete todos
- Mark todos as completed
- Hide completed tasks
- Add descriptions and due dates
- Data persistence using localStorage

## Technologies Used

- JavaScript (ES Modules)
- Web Storage API (localStorage)
- HTML5
- CSS3

## What I Learned

- Structuring applications using modules
- Separating logic, DOM manipulation, and storage
- Managing application state
- Persisting data with localStorage
- Reconstructing objects after JSON parsing
